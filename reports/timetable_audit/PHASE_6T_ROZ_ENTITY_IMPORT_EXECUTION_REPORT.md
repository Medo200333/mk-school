# Phase 6T - ROZ Entity Import Execution

## الهدف
تنفيذ استيراد كيانات ROZ المؤكدة فقط داخل قاعدة البيانات:
- المدرسون
- المواد
- الفصول

بدون استيراد حصص أو Slots أو CLASSTT mapping.

## مصدر التنفيذ
- Endpoint: `POST /api/v1/school-timetable/import/asctt-roz/entities`
- Frontend proxy: `POST /api/timetable/import/asctt-roz/entities`
- Confirm phrase: `IMPORT_ROZ_ENTITIES_ONLY`

## قيود الأمان
- التنفيذ بدون confirm رجع 409.
- `safe_to_import_slots=false`.
- لا يوجد استيراد للحصص في هذه المرحلة.
- ثاني Dry-Run بعد التنفيذ أثبت idempotency.
